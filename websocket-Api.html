🔐 Аутентификация

Все запросы к API требуют аутентификации через API ключ

Подключение с API ключом 
// WebSocket подключение
const API_KEY = 'cc_ws_xxxxxxxxxxxxxxxx';
const ws = new WebSocket(`wss://api.cycle-corp.com/ws?api_key=${API_KEY}`);

// HTTP запросы
const headers = {
    'X-API-Key': API_KEY,
    'Content-Type': 'application/json'
};
Важно: Никогда не публикуйте ваш API ключ в клиентском коде. Используйте серверную часть для безопасного хранения ключей.

💡 Примеры кода

JavaScript

class CycleCorpWS {
    constructor(apiKey) {
        this.ws = new WebSocket(
            `wss://api.cycle-corp.com/ws?api_key=${apiKey}`
        );
        this.setupEventHandlers();
    }

    setupEventHandlers() {
        this.ws.onmessage = (event) => {
            const data = JSON.parse(event.data);
            this.handleMessage(data);
        };
    }

    subscribe(channel) {
        this.ws.send(JSON.stringify({
            action: 'subscribe',
            channel: channel
        }));
    }
}

Python

import websocket
import json

class CycleCorpClient:
    def __init__(self, api_key):
        self.api_key = api_key
        self.ws_url = f"wss://api.cycle-corp.com/ws?api_key={api_key}"

    def connect(self):
        self.ws = websocket.WebSocketApp(
            self.ws_url,
            on_message=self.on_message
        )
        self.ws.run_forever()

    def on_message(self, ws, message):
        data = json.loads(message)
        print('Received:', data)

🎯 Best Practices

Совет: Следуйте этим рекомендациям для максимальной эффективности

Оптимальные практики

  • Автоматическое переподключение: Реализуйте логику повторного подключения при обрыве связи
  • Обработка ошибок: Всегда обрабатывайте возможные ошибки подключения и парсинга данных
  • Rate limiting: Соблюдайте лимиты запросов для избежания бана
  • Кэширование: Кэшируйте часто используемые данные для снижения нагрузки

Пример переподключения

JavaScript 
class RobustWebSocket {
    constructor(url) {
        this.url = url;
        this.connect();
    }

    connect() {
        this.ws = new WebSocket(this.url);

        this.ws.onopen = () => {
            console.log('✅ WebSocket connected');
        };

        this.ws.onclose = () => {
            console.log('Reconnecting...');
            setTimeout(() => this.connect(), 5000);
        };
    }
}
